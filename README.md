# ⚽ WC 2026 Daily WhatsApp Newsletter

Automated daily World Cup newsletter delivered to your WhatsApp — twice a day, every day of the tournament. Includes match recaps, predictions, and a live visual knockout bracket.

---

## What it does

| Run | Time (SGT) | Content |
|-----|-----------|---------|
| Morning | 8:00 AM | Yesterday's/early morning results + today's match predictions |
| Midday | 1:00 PM | Morning results + afternoon/evening match predictions |

Each message includes a link to the live visual bracket (hosted on GitHub Pages), updated with every run.

## One-time setup (~25 minutes)

### Step 1 — Fork/clone this repo

Create a new GitHub repo and upload these files, or fork this project.

### Step 2 — Enable GitHub Pages

1. Go to your repo → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: `main`, folder: `/docs`
4. Save. GitHub will give you a URL like `https://yourusername.github.io/wc-newsletter`

### Step 3 — Set up Twilio (WhatsApp)

1. Sign up at [twilio.com](https://twilio.com) (free)
2. In the Twilio Console, go to **Messaging → Try it out → Send a WhatsApp message**
3. Follow the sandbox activation: send the join code from your WhatsApp to their number (e.g. `+1 415 523 8886`)
4. Note down:
   - **Account SID** (starts with `AC...`)
   - **Auth Token**
   - **Sandbox number** (e.g. `whatsapp:+14155238886`)

### Step 4 — Get your API keys

- **Anthropic API key**: [console.anthropic.com](https://console.anthropic.com) → API Keys
- **Sportradar API key**: [developer.sportradar.com](https://developer.sportradar.com) → sign up for a free trial key for Soccer v4

### Step 5 — Add GitHub Secrets

Go to your repo → **Settings** → **Secrets and variables** → **Actions** → **New repository secret**

Add these 7 secrets:

| Secret Name | Value |
|-------------|-------|
| `ANTHROPIC_API_KEY` | Your Anthropic API key |
| `TWILIO_ACCOUNT_SID` | Your Twilio Account SID |
| `TWILIO_AUTH_TOKEN` | Your Twilio Auth Token |
| `TWILIO_WHATSAPP_FROM` | `whatsapp:+14155238886` (Twilio sandbox number) |
| `TWILIO_WHATSAPP_TO` | `whatsapp:+6591234567` (your number, with country code) |
| `SPORTRADAR_API_KEY` | Your Sportradar API key |
| `BRACKET_URL` | `https://yourusername.github.io/wc-newsletter/bracket.html` |

### Step 6 — Test it manually

1. Go to your repo → **Actions** → **WC 2026 Daily Newsletter**
2. Click **Run workflow** → **Run workflow**
3. Check the logs. You should receive a WhatsApp within ~30 seconds.

## Troubleshooting

**No WhatsApp received?**
- Check the Actions log for errors
- Make sure you completed the Twilio sandbox activation (send the join message from your WhatsApp)
- Verify `TWILIO_WHATSAPP_TO` includes `whatsapp:` prefix and full country code

**"Sportradar fetch failed" in logs?**
- The script falls back to mock data if Sportradar is unavailable — the newsletter still sends
- Check your Sportradar API key is active

**Bracket page shows "TBD" everywhere?**
- Normal before the knockout stage begins. It will populate once group stage ends.

## Schedule

The cron runs at:
- `0 0 * * *` UTC = 8:00 AM SGT
- `0 5 * * *` UTC = 1:00 PM SGT

To change times, edit `.github/workflows/wc-newsletter.yml` and convert your desired SGT time to UTC (SGT = UTC+8, so subtract 8 hours).
